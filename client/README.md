# 个人博客

## 运行前必备环境
`node` `mongodb`

## Project Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:4000
$ npm run start

# Compiles and hot-reloads for development
$ npm run serve

#Compiles and minifies for production
$ npm run build

```
